#🔹 Simple WPF MVVM Application

A lightweight desktop application built with WPF following the MVVM (Model-View-ViewModel) design pattern. The project demonstrates clean separation of concerns, maintainable architecture, and responsive UI binding.

Key features:

Clear MVVM structure (Model, View, ViewModel)
Data binding with INotifyPropertyChanged
Command handling using RelayCommand
Simple navigation and state management
Clean and scalable project structure

Tech stack:

.NET (WPF)
C#
MVVM Toolkit / custom MVVM implementation

Purpose:
This project serves as a foundational template for building scalable WPF applications with best practices in UI architecture and code organization.
